## FIAT

| Gateway | Value | Description |
| ------- | ----- | ----------- |
| PayPal | `PAYPAL` | Use PayPal as a payment gateway and its Alternative Payment Methods (APMs) |
| Stripe | `STRIPE` | Accept Credit Cards through Stripe |
| Skrill | `SKRILL` | Use your Business Skrill account |
| Perfect Money | `PERFECT_MONEY` | Use your connected Perfect Money account |
| Cash App | `CASH_APP` | Use the Sellix Cash App integration to accept Cash App payments |
